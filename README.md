## 📌 About the Project

**SkinSignal** is an AI-powered skin analysis platform that detects and classifies skin conditions directly from uploaded images using deep learning. A trained Convolutional Neural Network (CNN) analyses images at the pixel level to identify multiple skin signs and deliver automated, objective diagnostics — no dermatologist or human expertise required.

> 🔬 Decode your skin. Understand your health. Powered by AI.

---

## 🚨 Problem Statement

Skin conditions affect millions of people worldwide, yet access to professional dermatological diagnosis remains limited, expensive, and time-consuming. Most citizens have no quick, accessible way to identify their skin concerns. Existing solutions either require costly consultations or rely on inaccurate self-diagnosis.

---

## ✅ Solution

An intelligent skin analysis platform that:
- Accepts a skin image uploaded by the user
- Runs it through a trained CNN model for pixel-level classification
- Identifies multiple skin signs and conditions automatically
- Delivers objective diagnostics, treatment insights, and product recommendations

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- TensorFlow 2.x
- OpenCV
- Render account (for deployment)

### Installation

```bash
# Clone the repository
git clone https://github.com/khushi-urkude/skinsignal.git

# Navigate to project directory
cd skinsignal

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```

### Environment Variables

```env
MODEL_PATH=your_trained_model_path
API_KEY=your_api_key
PORT=5000
```
