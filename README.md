# ReviewPulse

ReviewPulse is a live SaaS MVP for clinics and specialty practices that helps teams automate review requests, manage enquiries, track patient follow-ups, and handle reply workflows from one dashboard.

## What it does

- secure practice login with Supabase auth
- review queue with draft replies and approval states
- patient list with review request tracking
- enquiry tracker with follow-up scheduling
- campaign history for outreach batches
- automation task list for reminders and callbacks
- practice settings and exportable workspace report

## Stack

- React + Vite
- Supabase Auth + Postgres
- Render deployment
