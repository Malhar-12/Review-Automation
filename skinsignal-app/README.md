# SkinSignal App

This folder contains the first real app scaffold for SkinSignal.

## What is included

- Vite + React app setup
- sidebar navigation
- dashboard view
- reviews view
- patients view
- campaigns view
- enquiries view
- settings view
- sample data for demo flows

## Run locally

1. Install dependencies with `npm install`
2. Start the app with `npm run dev`
3. Open the local URL shown by Vite

## Notes

- This version uses sample data only
- No backend is connected yet
- No auth is connected yet
- It is a good base for wiring Supabase, OpenAI, and WhatsApp later
