# SkinSignal App

This is the React MVP for SkinSignal, a review and enquiry management console for dermatology and aesthetic clinics.

## What is included

- Vite + React setup
- interactive dashboard, reviews, patients, campaigns, enquiries, and settings screens
- browser persistence for offline demo mode
- Supabase client wiring with graceful local fallback
- SQL schema file for the first database setup

## Run locally

1. Install dependencies with `npm install`
2. Copy `.env.example` to `.env.local` if needed
3. Start the app with `npm run dev`
4. Open the local URL shown by Vite

## Supabase setup

1. Open your Supabase project
2. Go to `SQL Editor`
3. Run the SQL in [supabase-schema.sql](./supabase-schema.sql)
4. Make sure `.env.local` contains:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`

When the tables exist, the app will sync there automatically. If they do not exist yet, the app stays usable in local demo mode.

## Notes

- This is still a single-tenant MVP
- Auth is not connected yet
- Review drafts are still local, not generated by OpenAI yet
- WhatsApp automation is still a planned next step
